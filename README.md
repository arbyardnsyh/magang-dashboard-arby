## Project Overview

This project is a responsive web dashboard that provides users with a modern banking and financial management experience. The interface is designed to closely follow the provided Figma design while maintaining clean code structure and reusable React components.

The dashboard includes several core features commonly found in modern financial applications, such as account overview, loan management, user settings, statistics visualization, and responsive navigation.
